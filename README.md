📊 HR Analytics Dashboard – Power BI Project
📌 Project Overview

The HR Analytics Dashboard is an interactive Power BI project designed to analyze employee data and understand key workforce metrics such as attrition, demographics, salary distribution, and department-wise insights.

The goal of this project is to help HR professionals and management make data-driven decisions to reduce employee attrition and improve workforce planning.

🗂️ Data Source

Dataset exported in CSV format

Employee data including demographics, job role, salary, education, department, and years at company

🧹 Data Cleaning & Preparation

Data cleaning was performed in Power BI using Power Query Editor:

Imported CSV file into Power BI

Removed duplicate records

Handled missing (null) values

Corrected and transformed data types

Created calculated columns and measures

Structured data for proper analysis

📈 Dashboard Features

The dashboard provides the following key insights:

🔹 Overall Metrics

Total Employees

Total Attrition

Attrition Rate (%)

Average Age

Average Salary

Average Years at Company

🔹 Attrition Analysis

Attrition by Gender

Attrition by Education

Attrition by Age Group

Attrition by Job Role

Attrition by Salary Slab

Attrition by Years at Company

🔹 Department-wise Analysis

Interactive slicers allow filtering by:

Human Resources

Sales

Research & Development

This helps in understanding attrition patterns across different departments.

🎯 Key Insights Delivered

Identified departments with highest attrition

Analyzed salary slabs contributing to employee turnover

Compared attrition trends across gender and education levels

Evaluated employee retention based on years at company

🛠️ Tools & Technologies Used

Power BI

Power Query

DAX (Data Analysis Expressions)

📊 Conclusion

This HR Analytics Dashboard enables organizations to monitor workforce trends, identify attrition drivers, and make strategic HR decisions. It demonstrates strong skills in data cleaning, data modeling, DAX calculations, and dashboard visualization.
